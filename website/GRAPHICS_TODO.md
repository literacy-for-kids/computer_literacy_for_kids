# Teacher Toolkit Development To-Do List

## Phase 1: Define the Product

First decide exactly what the teacher kit is.

**Goal:**
A downloadable package that helps teachers run the curriculum quickly with minimal prep.

**Tasks:**

- [ ] Write a short Teacher Toolkit description
- [ ] Decide pricing target ($15-$25)
- [ ] Decide final format:
  - [ ] printable PDF
  - [ ] slide deck
  - [ ] printable + digital bundle
- [ ] Choose a name

**Example:**

Computer Literacy for Kids  
Teacher Toolkit  
Printable Classroom Materials + Teaching Guides

## Phase 2: Core Teacher Guide

This is the most valuable piece.

Create a short teacher handbook (10-15 pages).

**Tasks:**

- [ ] Create sections:

### 1. Curriculum overview for teachers

- how the course works
- pacing
- required tools

### 2. Session structure guide

Explain:

- Guided Session 1
- Guided Session 2
- Independent Session

### 3. Classroom setup checklist

Include:

- computers ready
- typing website
- browser settings
- folder structure

### 4. Troubleshooting guide

Examples:

- Student can't find file
- Program won't open
- Internet search confusion

### 5. Teaching tips

Examples:

- encourage experimentation
- avoid giving answers immediately
- use reflection questions

## Phase 4: Student Worksheets

Worksheets help structure independent work.

Create simple one-page worksheets.

### Digital Safety Reflection

**Questions:**

- What should you do if something online feels strange?
- What is private information?
- What does "Talk it Out" mean?

### File Organization Worksheet

**Activity:**

- Draw folders and decide what files go in each

### Smart Searching Worksheet

**Activity:**

- Find a fact
- Write the source
- Explain why you trust it

### Coding Thinking Worksheet

**Activities:**

- Write steps for making a sandwich
- Circle the loop
- Identify conditions

### AI Thinking Worksheet

**Questions:**

- What can AI help with?
- Why should we check AI answers?
- What should we never share?

### Project Planning Sheet

Students plan their final project.

**Sections:**

- My Idea
- Characters / Topic
- Beginning
- Middle
- End
- Tools I will use

## Phase 5: Classroom Tools

These are small items teachers love.

### Debugging Checklist

Poster or card:

- Did you save your work?
- Did you click the right program?
- Try again
- Close and reopen
- Ask for help

### Typing Progress Chart

Simple printable tracker.

Students mark:

- Lesson completed
- Stars earned
- Speed improvement

### Project Rubric

Teachers can evaluate final projects.

**Categories:**

- Creativity
- Effort
- Understanding
- Presentation

### Achievement Certificates

Printable certificates:

- Digital Explorer
- Coding Beginner
- Internet Safety Hero
- Computer Creator
