# Independent Session Setup Tips

These tips will help you create a productive and enjoyable independent session experience for your learner.

---

## 1. Create Visual Instruction Cards

Simple step-by-step guidance with pictures for each week's independent task. Use screenshots or drawings to show exactly what the child should do.

## 2. Set Up a Visual Timer

Helps the child manage the 20 minutes independently. A countdown timer they can see builds time-management awareness and keeps them on track.

## 3. Prepare a "Help Card"

Include 2–3 common troubleshooting tips relevant to the week's tools:
- "How to Save"
- "Undo Button"
- "Check Wi-Fi"
- "Ask for Help if Stuck for 5 mins"

Update this card as new tools are introduced each week.

## 4. Create an Achievement Tracker

A chart (themed to the child's interests) where they can place stickers after completing independent work. This adds motivation and provides a visual record of progress.

## 5. Establish a "Show and Tell" Routine

After each independent session, spend 1–2 minutes letting the child briefly share what they created, discovered, or found challenging. This reinforces learning and builds communication skills.
