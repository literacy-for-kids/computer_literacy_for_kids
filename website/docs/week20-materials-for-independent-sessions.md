# Materials for Independent Sessions

Keep these materials prepared and accessible for each week's independent session.

---

## Essentials

| Material | Purpose |
|----------|---------|
| **Printed instruction cards** | Step-by-step visual guides specific to each week's task |
| **Troubleshooting guide ("Help Card")** | Common fixes with pictures for each tool |
| **Achievement chart & stickers** | Track progress and motivate learners |
| **Visual timer** | Help the child manage their 20-minute session |

## Supplementary Materials

- **Pre-loaded templates or starter files** — Use when appropriate (e.g., document templates, Scratch starters, presentation files to add to)
- **Quick-reference sheets** — For commonly used tools:
  - Paint 3D toolbar guide
  - Scratch block categories overview
  - TinkerCAD view controls
  - Keyboard shortcut cheat sheets

:::tip
Update your Help Cards and quick-reference sheets each time a new tool is introduced. Keeping materials current reduces frustration and builds independence.
:::
