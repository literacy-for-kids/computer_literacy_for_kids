# Executive Function Skills for Digital Projects

Computer projects involve files, folders, logins, steps, debugging, versions, and saving work. Executive function helps kids organize digital tasks and recover when they lose track.

This page is the local doorway into the Literacy for Kids [Executive Function Toolkit](https://www.literacy-for-kids.com/docs/toolkits/executive-function), connected to the digital skills this curriculum builds.

## A few core ideas

- **Digital work needs visible steps.** Steps hidden in your head are easy to drop.
- **File organization reduces future confusion.** A clear name today saves a search tomorrow.
- **Debugging works better with checklists.** A short list keeps you from checking the same thing twice.
- **Saving and restart points help projects survive interruptions.** A bookmark beats redoing work.

## When this shows up

- When files are hard to find
- When a project has many steps
- When code or Scratch projects need debugging
- When too many tabs or tools are open
- When a project is interrupted
- When final-project work needs tracking

## Tools that help

- **File checklist** — name it, save it, put it where you can find it.
- **Debugging checklist** — the same ordered checks each time.
- **Restart bookmark** — "I stopped at ___, next step is ___."
- **One-tab / one-tool focus setup** — fewer attention pulls.
- **Done-enough check** — is this version good enough for the purpose?

:::tip Executive Function Moment
Before you leave a digital project, make a restart bookmark: "I stopped at ___, and the next step is ___." Future-you will find the project faster.
:::

:::caution Everyday task skills, not a diagnosis
These are everyday planning and task-management tools, not therapy, medical advice, disability evaluation, ADHD treatment, or a replacement for trusted adults, educators, caregivers, or qualified support. Kids should not be treated as lazy or broken when starting, focusing, organizing, or finishing is hard. If tasks feel consistently overwhelming, inaccessible, unsafe, or severely distressing, involve a trusted adult and appropriate support.
:::
